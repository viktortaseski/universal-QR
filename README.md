# Universal QR Code Generator

A simple Python script that generates a QR code image from a URL.

## Requirements

- Python 3.x
- `qrcode` library

## Installation

```bash
pip install qrcode[pil]
```

## Usage

1. Open `generator.py` and set the `URL` variable to your desired link.
2. Run the script:

```bash
python generator.py
# or
python3 generator.py
```

The QR code will be saved as `qr.png` in the same directory.
